# HTML/CSS Final Project - Design Refinement

## Overview
This repository contains my final project submission for an HTML/CSS course.

The project demonstrates:
- Semantic HTML structure
- Core CSS styling and layout organization
- Typography customization with Google Fonts
- Basic accessibility practices (focus states, ARIA labels, keyboard support)

The main page is a refined static website layout, supported by two additional practice/demo files focused on accessibility and interaction patterns.

## Project Goals
- Build a clean, readable webpage using semantic HTML.
- Apply consistent visual styling through an external CSS stylesheet.
- Improve usability with clear typography, spacing, and color contrast.
- Include accessibility-minded markup and interaction patterns.

## Technologies Used
- HTML5
- CSS3
- Google Fonts (Playfair Display)
- Vanilla JavaScript (used in one demo page)

## Folder Structure

```text
unit 14/
```

## File Descriptions
- index.html: Main final project page showing the refined layout and styling.
- css/style.css: Shared stylesheet for the main page; includes reset-style rules, typography, layout spacing, and focus styles.
- example.html: Small accordion interaction demo showing progressive enhancement and accessibility-related concepts.
- test.html: Basic accessibility test page that includes button and ARIA label usage.

## Accessibility Considerations
This project includes several accessibility-focused improvements:
- Language declaration on HTML documents (`lang="en"`)
- Meaningful heading structure
- Keyboard focus visibility using `:focus` outlines
- ARIA usage examples (`aria-label` and `role="button"`)
- Alternative text on images

## How to Run
Because this is a static project, no build tools are required.

1. Clone or download this repository.
2. Open the project folder in VS Code.
3. Open `index.html` in your browser.
   - You can use VS Code Live Server, or
   - Double-click the file to open it directly.
